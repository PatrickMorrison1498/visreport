About Vis.Report

Vis.Report contains a databse of visability and conditions reports for dive sites around Perth, all contributed by users. Dive sites are curated manually. If you have a suggestion for a site that isn't on here, please email secretary@maawa.net. Please share it around, the more people that contribute the more useful the site becomes!

It is built on the Django webframework, connected to a PostgeSQL database with a PostGIS extension. All passwords are stored as a secure hash using Django's authentication system. All code can be found on the Github page.
