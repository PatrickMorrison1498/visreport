About Vis.Report

Vis.Report contains a database of visibility and condition reports for dive sites around Perth, all contributed by users. Dive sites are curated by myself. If you have a suggestion for a site that isn't on here, please email secretary@maawa.net. Please share it around, the more people that contribute frequently, the more useful the information becomes!

The data contributed is made freely available for research. The factors affecting dive conditions and visibility are not yet fully understood, particularly in the Swan River. This sort of information from citizen scientists will give us greater insight, and make for better diving. JSON files for the sites and reports are dynamically generated straight from the database. These contain exactly the same information as is visible on the site, except username is replaced with a unique number.

This site is built on the Django webframework, connected to a PostgeSQL database with a PostGIS extension. All passwords are stored as a secure hash using Django's authentication system. All code can be found on the Github page.
